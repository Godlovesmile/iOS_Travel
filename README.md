#Travel
##项目介绍:
- 基于对于技术的好奇心,自学点iOS开发,将自己学习的技术点转化为实际,仿写旅游App,加入一些自己的独特想法.
	
##项目技术点
- UITableView+UICollectionView页面布局;
- 基于AFNetworking做网络请求;
- XMPP实现即时通讯;
- CoreData数据本地持久化
